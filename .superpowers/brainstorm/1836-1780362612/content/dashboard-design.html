Dashboard layout breakdown

1

Header Row

Title + timestamp on the left. Action buttons on the right: subtle "Refresh", primary "Apply Safe Tune" (accent blue), cautionary "Undo All" (red text). Clear visual hierarchy.

2

Health Banner

WinUI InfoBar-style strip. Green dot + "Good" text + large score number. Right side shows quick stats (active optimizations, undoable changes). Color shifts green → yellow → red by score.

3

Metric Cards (5-column grid)

CPU (blue), Memory (green), GPU (amber), Disk (purple), Network (pink). Each: label + secondary info, large monospace value, thin progress bar. Clickable → navigates to related category page.

4

Charts Row (2:1 split)

Left: Syncfusion SfCartesianChart with 60-second rolling area chart (CPU/Memory/GPU overlaid). Right: Per-core CPU vertical bars in a grid, collapsible "show all cores".

5

Bottom Row (1:1 split)

Left: Top processes table (sortable by CPU or Memory, with "End Process" option). Right: Disk + Network I/O bars with live throughput values.
