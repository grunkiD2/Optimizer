Category page anatomy

1

Page Header

Category icon + name + "3 of 5 active" summary. Right side: "Apply All" (accent) and "Undo Category" (subtle) — scoped actions for just this category.

2

Local Metrics Strip

Only the metrics relevant to this category. Performance → CPU + Memory + Power Plan. Network → throughput + latency. Storage → disk usage + temp file size. Inline mini sparklines.

3

Optimization Cards

Each optimization is a card with: icon, human-readable name, one-line description, status badge (ACTIVE/inactive), and a toggle switch. Expandable detail panel shows registry changes, impact (pros ▲ / cons ▼), and requirement badges (Reversible, Requires Admin, Requires Restart).

4

Visual states

Active cards: brighter text, blue icon background, green ACTIVE badge, blue toggle. Inactive: dimmed text, gray icon, gray toggle. Clear at a glance which optimizations are on.
